void CO2Trigger ( )

Live Data Simple request to CO2 sensor.

Note:
We use the fact that uC UART FIFO is 16 bytes long and we are sending this 7 byte message periodically after a long period of time (from UART's point of view) to achieve maximum performance without using DMA or interrupts.
