Detailed Description

Expands http://mbed.org/handbook/Serial with non-blocking functionality.

Author:
Orestis Zachariadis
Note:
Both putc and printf() use serial_putc() in serial_api.c.

Transmit FIFO size is 16. If we transmit new data and FIFO is full the new data gets lost. Same goes for Receive FIFO according to user manual (Overrun Error).

UART peripheral starts to transmit immediately after feeding it. If we don't feed with new data fast enough and transmit buffer gets empty we have a THRE interrupt.

Required time to transmit n bytes is (1/Baud)*(1+8+1)*n
